Fokus: Eksperimen plugin system, integrasi cloud build awal, feedback user, bugfix mayor.

- [ ] Integrasi feedback user untuk plugin/cloud

- [ ] Perbaikan bug dan stabilisasi
    - Bugfix mayor dari hasil feedback
